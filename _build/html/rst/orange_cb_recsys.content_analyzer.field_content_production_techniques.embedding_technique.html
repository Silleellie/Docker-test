Class that implements the abstract class EmbeddingSource. This class loads the embeddings from a binary file in a way that depends from embedding_type.

Parameters

  • file_path (str) – path for the binary file containing the embeddings

  • embedding_type (str) – Name of the technique used to learn the embedding that is being loaded the possible values are: “word2vec”, “doc2vec”, “fasttext”, “ri”
